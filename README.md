# Aethelgard-Re-Climate-Risk-Analytics
Data Science Project Lifecycle Coursework - Climate Risk Analytics for Aethelgard Re
## Project Overview
This project investigates the relationship between greenhouse gas emissions and global temperature anomalies for the fictitious reinsurance company Aethelgard Re.
## Objectives
Exploratory Data Analysis
Correlation Analysis
Predictive Modelling
10-Year Temperature Forecasting
Business Recommendations
 ## Tools Used
Python
Google Colab
Pandas
